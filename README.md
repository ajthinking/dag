# Data Story - Core

See also [data-story-org/gui](https://github.com/data-story-org/gui)

## Development

### Installation

```
yarn
# Build web files
tsc --watch
```

### Tests

Run tests with

```
yarn test --watch
```

### Add node

You may run the following command

```
yarn add-node YourNodeName
```

This will create a new `Node` class along with a test stub.
